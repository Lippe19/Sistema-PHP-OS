# Sistema-PHP-OS
Sistema PHP de ordem de serviços com landing page.

O sistema tem as seguintes funcionalidades:
- CRUD clientes 
- CRUD ordem de serviços 
- CRUD usuários (administrador e funcionário)
- Backup do banco de dados 
- Métricas
